# DevPulse - Issue Tracker API

## 🔗 Live URL

[dev-pulse-steel-nine.vercel.app](https://dev-pulse-steel-nine.vercel.app/)

---

## 📌 Project Overview

A collaborative backend API for software teams to report bugs, suggest features, and manage issue resolution with role-based access control.

---

## 🚀 Features

- User registration (contributor / maintainer)
- JWT-based authentication
- Create, read, update, delete issues
- Role-based access control
- Filter issues by type and status
- Sort issues by newest/oldest
- Reporter auto-assigned from logged-in user
- Secure protected routes with middleware

---

## 🛠 Tech Stack

- Node.js
- Express.js
- TypeScript
- neonDB
- JSON Web Token (JWT)
- bcrypt
- pg (node-postgres)
